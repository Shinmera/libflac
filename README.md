## About libflac
This is a very minimal library to decode FLAC files.

## API
The API is documented in [flac.h](src/flac.h).

## Building It
Requirements:

* CMake 3.1+
* A C99 compiler

Steps:

* `mkdir build`
* `cd build`
* `cmake ..`
* `make`
